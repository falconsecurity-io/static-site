# Static Site Generator

This project follows a course from [Boot.dev](https://boot.dev). The goal is to build a static site generator from scratch.

## Course Objective

Build a tool that converts raw content (Markdown, images) into a static website (HTML, CSS). Deploy the site to a live URL.

## What Is a Static Site Generator?

A static site generator transforms content files into a website. Static sites are fast, secure, and easy to host.

Examples of use cases:
- Blogs
- Portfolios
- Landing pages
- Documentation
